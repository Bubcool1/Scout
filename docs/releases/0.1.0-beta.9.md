# Scout 0.1.0-beta.9

Scout beta.9 adds an optional, evidence-led CV interview and an enforceable natural-voice review without taking control away from the user.

## Highlights

- Recommends Google XYZ for genuine achievement bullets while allowing the option to be switched off for each CV.
- Asks one targeted evidence question at a time, explains the intended bullet, and lets every question be skipped or the interview finished early.
- Accepts truthful qualitative outcomes when no metric exists and continues to prohibit invented claims or numbers.
- Offers an optional separate natural-voice pass for clichés, repeated openings, long bullets and locale mismatches.
- Records confirmed answers and exact bullet provenance in the private application folder.
- Adds a hash-bound CV quality report so edits invalidate stale reviews and draft overrides.
- Keeps previews available, blocks known unsupported claims and rendering failures, and requires explicit confirmation before downloading a draft with writing warnings.
- Lets useful confirmed answers be reviewed before optional promotion into the master CV.

There is no workspace schema migration and no new network service. Existing tailored CVs appear as legacy drafts and can be reviewed or explicitly downloaded unchanged. Application updates continue to preserve the separate private Scout workspace.
